# Biomedical Signal Processing – MATLAB

MATLAB scripts for analysis of physiological signals including ECG, EEG, and SpO2.
Covers spectral analysis, artifact detection and removal, and clinical feature extraction.

## Contents

| Script | Topic |
|--------|-------|
| `ecg_psd_analysis.m` | ECG power spectral density – Blackman window, Welch method |
| `eeg_artifact_removal.m` | EEG artifact detection and removal – Notch filter, bandpass filter |
| `spo2_spectral_analysis.m` | SpO2 spectral analysis – median frequency, spectral entropy, respiratory band power |

## Technologies
- MATLAB
- ECG, EEG, SpO2 signal processing
- Spectral analysis, digital filtering, Welch PSD
